### Notes on Configuration
**Path:** `$HOME/.config`  

**Description:** Central directory for most application configs.

**Notes/Usage:**  
- The setup is modular. You may need to adjust `$HOME` paths.  
- Some configs are overwritten by external tools (e.g., **nwg-look**). Do not edit those manually.

```bash
├── .config
│   ├── cava
│   ├── cmatrix
│   ├── copyq
│   ├── gtk-2.0
│   ├── gtk-3.0
│   ├── gtk-4.0
│   ├── htop
│   ├── hypr
│   ├── kitty
│   ├── nvim
│   ├── nwg-look
│   ├── rofi
│   ├── swaync
│   ├── Thunar
│   ├── wallust
│   ├── waybar
│   ├── wlogout
│   ├── xsettingsd
│   └── zed
```

**Description:** Visual audio spectrum analyzer.

**Notes/Usage:**  
- Adjust equalizer, smoothing, bars, and output style.  
- Create `~/.config/cava/shaders/` to implement custom shaders.

### cmatrix
**Path:** `~/.config` (no dedicated config by default)

**Description:** Matrix-style terminal screensaver.

**Notes/Usage:**  
- No config file required; use runtime flags.  

<details>
<summary>Command Examples</summary>

```bash
# Slow / fast
cmatrix -s
cmatrix -f

# Custom delay (microseconds)
cmatrix -u 50000

# Colors
cmatrix -C red
cmatrix -C blue
cmatrix -C yellow
cmatrix -C white
cmatrix -C cyan
cmatrix -C magenta

# Character sets
cmatrix -u   # Uppercase only
cmatrix -l   # Lowercase only
cmatrix -n   # Numbers only
cmatrix -o   # Katakana
cmatrix -a   # All available characters (default)

# Formatting
cmatrix -B   # Bold
cmatrix -b   # No bold

# Special
cmatrix -L   # Lambda mode
```
</details>

- Do not edit manually; values are overwritten.

- Generated by **nwg-look** and read by GTK apps.  
- Do not edit manually.

**Notes/Usage:**  
- Generated by **nwg-look**.

### hypr
**Path:** `~/.config/hypr/`  

**Description:** **Hyprland** window manager configuration.

**Notes/Usage:**  
- Config files are sourced by `hyprland.conf`. User defaults, such as your default file manager and terminal, are defined in these files.
- Controls layouts, animations, rules, input, and window behavior.

---

### kitty
**Path:** `~/.config/kitty/`

**Description:** GPU-accelerated terminal emulator.

**Notes/Usage:**  
- Configure fonts, colors, keymaps, performance flags in `kitty.conf`.  
- Optional: `~/.config/kitty/kitty-themes` for modular configs.

---

### nvim
**Path:** `~/.config/nvim/`

**Description:** Neovim text editor.

**Notes/Usage:**  
- Manage plugins, LSP, Treesitter, and keymaps via `init.lua` or `lua/`.  
- Keep `after/` for local overrides.

### nwg-look
**Path:** `~/.config/nwg-look/`

**Description:** GTK theme manager.

**Notes/Usage:**  
- Applies GTK themes and writes configs to `gtk-2.0/`, `gtk-3.0/`, `gtk-4.0/` and `xsettingsd`.  
- Re-run after theme changes.

### rofi
**Path:** `~/.config/rofi/`

**Description:** Application launcher and switcher.

**Notes/Usage:**  
- Customize `config.rasi` and theme files.  
- Set modi (e.g., `drun`, `window`) and keybindings.

---

### swaync
**Path:** `~/.config/swaync/`

**Description:** Wayland notification daemon.

**Notes/Usage:**  
- Configure layout, urgency rules, and styling in `config.jsonc` and `style.css`.  
- Supports images for notification bodies.
- bind `swaync-client -t -sw` in `~/.config/hypr/configs/keybinds.conf` to open notification center.
---

### Thunar
**Path:** `~/.config/Thunar/`

**Description:** XFCE file manager.

**Notes/Usage:**  
- Configure custom actions, templates, and bookmarks.

---

### wallust
**Path:** `~/.config/wallust/`

**Description:** Wallpaper-based color scheme generator.

**Notes/Usage:**  
- Generates color variables for other apps.  
- Integrate with Waybar, Rofi, and GTK by sourcing exported files.

---

### waybar
**Path:** `~/.config/waybar/`

**Description:** Wayland status bar.

**Notes/Usage:**  
- Configure modules in `config` and styles in `style.css`.  
- Common modules: workspaces, clock, volume, network, battery.

---

### wlogout
**Path:** `~/.config/wlogout/`

**Description:** Logout menu.

**Notes/Usage:**  
- Define layout and actions in `layout`.  
- Style with `style.css`.
- Bind in `~/.config/hypr/configs/keybinds.conf`

---

### xsettingsd
**Path:** `~/.config/xsettingsd/`

**Description:** X11 settings daemon for XWayland apps.

**Notes/Usage:**  
- Set fonts, themes, and toolkit hints for X apps under Wayland.
